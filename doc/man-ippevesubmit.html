ippevesubmit(1)

Name

ippevesubmit - prepare a submission for self-certification results.

Synopsis

ippevesubmit [ --help ]
ippevesubmit [ -m models.txt ] [ -o filename.json ] [ -p "product family" ] [ -s ] [ -u URL ] [ -y ] "Printer Name"

Description

ippevesubmit validates the results from the IPP EverywhereTM Printer Self-Certification Tools and generates a JSON file suitable for submission to the IEEE-ISTO Printer Working Group.

Options

The following options are recognized by ippevesubmit:
--help
Shows program help.
-m models.txt
Specifies a file containing a list of printer models, one per line, that are being submitted for certification.
-o filename.json
Specifies a JSON output file. If not specified, the JSON file is sent to the standard output.
-p "product family"
Specifies the product family for the models being submitted. If not specified, ippevesubmit will ask for the family name.
-s
Specifies that the submission is for a print server product and not a standalone printer.
-u URL
Specifies a product web page for the models being submitted. If not specified, ippevesubmit will ask for the web page URL. -y Specifies that the submitter used the tools from the PWG web site, has production-ready firmware on the printer or production-ready software on the print server, and that all output printed correctly. If not specified, ippevesubmit will ask the user to confirm each of these things.
